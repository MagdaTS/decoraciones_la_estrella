# Catalogs Section

Adds a Catálogos section to the single-page site, positioned between Cases and Contact. Displays three downloadable product catalogs (Día de Muertos, Fiestas Patrias, Navideño) in a responsive 3-column grid with cover cards, desktop hover preview, and PDF download/view buttons. Includes a one-time script extension to generate page preview WebPs and a new nav link.
